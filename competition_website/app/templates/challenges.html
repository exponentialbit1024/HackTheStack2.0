Buffer Overflows

Each buffer overflow attack is located inside a directory named "overflowN" (where 'N' is the number of the attack). The directory contains one or more C source files as well as one or more binary files. For each attack, one of the source files contains a call to a function 'get_pwd'. You must exploit the binary(s) given to cause that function to run and output the password for that level. Enter that password in the fields below for every challenge. There are 5.

Forensics Challenges

Each forensics challenge is located in a self contained directory. While these are not typical "attacks" or "exploits", there is some data hidden (either cryptographically or steganographically) in the files within that directory. Your task is to retrieve that data and extract a password from it.
